== Description ==

This shows you a different positive affirmation on every page of the admin section.

Positive affirmations are known to have positive psychological effects, including mood improvement and self-esteem boosts. Now, you can have it conveniently come up while you work on your WordPress website!

== Installation ==

1. Log into your WordPress admin section and then go to Plugins > Add New.
2. Click Choose File, and then choose the positive-affirmations.zip file.
3. Click Install Now.
4. Once installed, click Activate.
5. Enjoy!
